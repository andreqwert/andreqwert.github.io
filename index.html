Некоторые команды терминала Linux

touch example.py - создать файл example.py
> example.py - создать файл example.py
mkdir - создать папку
cd - переход по директориям
ls - показать файлы в директории

vim example.py - открыть файл в vim
:q - выход из vim без сохранения
:wq - выход из vim с сохранением
