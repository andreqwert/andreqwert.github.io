1. Линейный поиск (Linear search) 
    

    # A - массив
    # x - искомое значение

    def linear_search(A, x):
        """Linear search"""

        answer = 'NotFound'
        for i in range(0, len(A)):
            if A[i] == x:
                answer = i
        return answer


    linear_search(['A', 'B', 'C', 'D'], 'E') # Not Found
    linear_search(['A', 'B', 'C', 'D'], 'A') # 0
    linear_search(['A', 'B', 'C', 'D'], 'C') # 2
2. Рекурсия (Recursion) 
    
    def factorial(n):
        if n == 0:
            return 1
        else:
            return n*factorial(n-1)


    print(factorial(5)) # 120
3. Бинарный поиск (Binary search) 
    
    # Предполагается, что значения в массиве - в порядке возрастания!
    
    def binary_search(A, x):

        lower_bound = 0
        upper_bound = len(A)
        while lower_bound != upper_bound:
            compared_value = (lower_bound + upper_bound) / 2
            if x == A[compared_value]:
                return compared_value
            elif x < A[compared_value]:
                upper_bound = compared_value
            else:
                lower_bound = compared_value + 1
        return 'Not Found'


    binary_search([1, 2, 3, 4, 5, 6], 3)       # 2
    binary_search(['A', 'B', 'C', 'D'], 'D')   # 3
    binary_search(['A', 'B', 'C', 'D'], 'F')   # Not Found
Бинарный поиск проходит имеет следующий алгоритм: Пусть изначально есть массив чисел: 1 3 5 7 10 19 21 middle = 7, lower_bound = 1, upper_bound = 21. Ищем число 21. 7 < 21 --> сдвигаемся вправо; 10 < 21 --> сдвигаемся вправо; 19 < 21 --> сдвигаемся вправо; 21 == 21. ОК. Аналогично можно найти, например, число 5. 7 > 5 --> сдвигаемся влево; 5 == 5.
